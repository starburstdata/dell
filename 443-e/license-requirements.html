    Starburst Enterprise license#

    As mentioned in the overview, a license for Dell Data Analytics Engine, powered by Starburst Enterprise platform (SEP) includes support and enables numerous features.

    To purchase a license or obtain a free trial, contact us or email us at hello@starburstdata.com.

    License usage#

    After receiving a signed license file from Starburst Enterprise, it needs to be stored on all SEP nodes in your clusters.

    The license file needs to be named starburstdata.license and located within the SEP installation directory as etc/starburstdata.license, or whatever directory is configured as the etc directory with the launcher script.

    Kubernetes users need to follow the available instructions, which are included in the complete documentation for Kubernetes usage with Helm.

    Starburst Enterprise Elite license#

    The Starburst Enterprise Elite license unlocks premium features, such as Starburst Stargate and Enable Starburst Warp Speed for your cluster, in addition to the support and features included in the Starburst Enterprise license.

    Managing and using the licenses is identical.
