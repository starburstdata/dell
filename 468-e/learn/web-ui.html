    Explore the web UI#

    The Dell Data Analytics Engine, powered by Starburst Enterprise platform (SEP) web UI is the web interface you use to access a SEP cluster. To access the web interface you must have a valid license configured.

    The Starburst Enterprise web UI contains many features including, Insights, data products, domain management, and the query editor.

    Read more information in the Feature tabs section.
