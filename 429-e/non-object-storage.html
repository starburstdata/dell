    Non-object storage connectors#

    This chapter describes the client connectors available in Starburst Enterprise to create catalogs that work with a data source’s native client libraries to execute queries on different data sources.

    Dell Data Analytics Engine, powered by Starburst Enterprise platform (SEP) includes numerous additional and enhanced connectors.

    Connector types#

    Starburst Enterprise connects to a variety of data source types, including object storage, RDBMSs, streaming data, and more. Find the connector for your data source below:
