    Helm chart release notes#

    Each Dell Data Analytics Engine, powered by Starburst Enterprise platform (SEP) release is accompanied by a new release of the supported Helm charts. Details of the changes on each chart can be found in the following release note sections.

    429.0.1 (29 Nov 2023)#

    • Update to SEP 429-e.0.

    • Upgrade base image from RHEL 8 to Red Hat UBI 9 micro.

    • Change SEP launcher language from Python to Go to reduce potential CVE exposure.

    423.7.0 (27 Nov 2023)#

    • Update to SEP 423-e.7.

    • Improve liveness probe in license-verified container in EKS marketplace deployments.

    413.12.0 (27 Nov 2023)#

    • Update to SEP 413-e.12.

    423.6.0 (13 Nov 2023)#

    • Update to SEP 423-e.6.

    • Update to SEP 423-e.5.

    • Update to SEP 423-e.4.

    • Update to SEP 413-e.10.
